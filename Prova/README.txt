Microserviço RPC para contabilizar palavras em um dicionário

Aluno: Vinicius Assumpcao de Araujo
Matrícula: 200028472

Instruções de execução:


1. Executar o servidor:
   $ ./dictionary_server <porta>

2. Em outro terminal, executar o cliente no modo inclusão:
   $ ./dictionary_client <endereço_servidor> 0 <arquivo_entrada>
